Welcome: 
Instructions: 
1. Download and install Visual Studio (https://go.microsoft.com/fwlink/?LinkId=691978&clcid=0x409)
2. Download and install Unity 5.3.3 (https://unity3d.com/get-unity/download?ref=personal)
3. Download and install UnityVS: http://unityvs.com/
4. Import the New Unity Project(2D-Platformer) project in Unity
5. Run the Voice_Control.exe from the Voice_Control(plugin) project 
(Make sure your Windows Speech Recognizer is set and running before this step)
6. Click on Mic icon to start listening
7. Start the game and enjoy

Please refer to Demo video to see working

Supported Platform: Windows, Tested on Windows 10(64 bit)
